## 🖥️ Como rodar este projeto 🖥️


### Execução:

1. Clone este repositório:

   ```sh
   git clone git@github.com:tricia-sz/booking-hotel.git
   ```

2. Acesse o diretório do projeto:

   ```sh
   cd booking-hotel
   ```



6. Abra o arquivo html  na pasta local e abra no navegador.
